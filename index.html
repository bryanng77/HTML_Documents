Programming Languages

These are the programming languages that I'm familiar with!

Structured Query Language (SQL)

Structured Query Language, abbreviated as SQL, is a domain-specific language used in programming and designed for managing data held in a relational database management system, or for stream processing in a relational data stream management system.

Python

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation via the off-side rule. Python is dynamically typed and garbage-collected.

R-Programming

R is a programming language for statistical computing and graphics supported by the R Core Team and the R Foundation for Statistical Computing.
